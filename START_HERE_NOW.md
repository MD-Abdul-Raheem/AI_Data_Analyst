## Troubleshooting

### Server won't start?
```bash
pip install flask pandas numpy matplotlib seaborn scipy google-generativeai
```

### Port 5000 busy?
- Close other apps using port 5000
- Or edit `app_final.py` line 217: change `port=5000` to `port=5001`

### AI insights not working?
- Check internet connection
- Gemini API requires internet
