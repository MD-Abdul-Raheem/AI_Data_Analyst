## Troubleshooting

### App won't start?
```bash
# Clear cache and reinstall
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### Analysis not working?
- Check browser console (F12) for errors
- Verify internet connection (API requires internet)
- Try with smaller dataset first
- Check if API key is valid

### No results showing?
- Wait 20-30 seconds for large datasets
- Check the browser console for API errors
- Try refreshing the page

## Tips for Best Results

1. **Clean Data**: Remove special characters from column names
2. **Reasonable Size**: 100-10,000 rows works best
3. **Clear Headers**: Use descriptive column names
4. **Mixed Data**: Include both numeric and categorical columns
5. **Date Columns**: Format dates consistently
